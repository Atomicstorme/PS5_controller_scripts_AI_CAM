PS5 Controller Scripts
======================

FIRST TIME SETUP
----------------
Install the ViGEmBus driver (required, one-time):
https://github.com/ViGEm/ViGEmBus/releases

Download ViGEmBus_Setup_x64.msi, run it, restart PC.

USAGE
-----
1. Connect PS5 controller (USB or Bluetooth)
2. Run PS5ControllerScripts.exe
3. Enable scripts you want from the Scripts panel
4. Play your offline games!

INCLUDED SCRIPTS
----------------
- anti_recoil.lua   - Compensates for weapon recoil
- auto_sprint.lua   - Auto-sprint when pushing forward
- aim_assist.lua    - Slows aim when ADS
- rapid_fire.lua    - Rapid fire for semi-auto
- deadzone.lua      - Adjustable stick deadzones
- _template.lua     - Template for custom scripts

For OFFLINE/SINGLE-PLAYER games only.
